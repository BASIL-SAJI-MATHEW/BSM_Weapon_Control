# Security Policy

## Supported Versions

Currently, only the latest release of the **BSM Weapon Control System** receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerabilities, client-to-server trigger exploits, or SQL injection loopholes within this resource, please **do not report it via public GitHub issues.**

Instead, please privately contact **Basil Saji Mathew (BSM)** directly via Discord or associated contact means provided on the author's official channels.

**What to include in your report:**
- Description of the vulnerability.
- Step-by-step reproduction instructions.
- The framework you are testing on (ESX/QB/etc.).
- Any potential fix or insight you might have.

We take exploits very seriously (especially considering this script manages player inventories and combat elements) and will prioritize pushing a hotfix immediately.
